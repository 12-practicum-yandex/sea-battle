# sea-battle

## Команды

Запуск проекта

```sh
npm run start
```

Сборка production версии

```sh
npm run build
```

Скрипт для инициализации husky

```sh
npm run prepare
```

Скрипты линтеров:

- проверка файлов по правилам eslint

```sh
npm run link
```

- фикс файлов по правилам eslint

```sh
npm run lint-fix
```
